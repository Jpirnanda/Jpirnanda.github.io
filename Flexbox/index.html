Guia Prático do CSS Flexbox

1. O Container Pai

Para começar, aplicamos display: flex no elemento pai. Isso transforma os filhos em itens flexíveis que ficam lado a lado.

2. Direção (Flex Direction)

O padrão é em linha. Podemos mudar para coluna se quisermos empilhar os itens.

Column (Coluna)

.container {
  display: flex;
  flex-direction: column;
}

3. Alinhamento Horizontal (Justify Content)

A propriedade justify-content controla como os itens são distribuídos na horizontal (esquerda/direita).

Center (Centro Horizontal)

.container {
  display: flex;
  justify-content: center;
}
1
2
3

Space-Between (Espaço Entre)

Cola o primeiro item na esquerda, o último na direita, e divide o resto.

.container {
  display: flex;
  justify-content: space-between;
}

Space-Around (Espaço ao Redor)

Distribui espaço igual ao redor de cada item. Note que o espaço nas pontas é metade do espaço entre os itens.

.container {
  display: flex;
  justify-content: space-around;
}

Space-Evenly (Espaço Uniforme)

Distribui o espaço de forma perfeitamente igual, inclusive nas bordas e entre os itens.

.container {
  display: flex;
  justify-content: space-evenly;
}

4. Alinhamento Vertical (Align Items)

A propriedade align-items controla como os itens se comportam na vertical (topo/fundo).

Flex-Start (Topo / Início)

Alinha os itens no topo do container. (Útil para evitar que eles estiquem).

.container {
  display: flex;
  /* Altura necessária para ver o efeito */
  height: 200px;
  align-items: flex-start;
}
1
2
3

Center (Centro Vertical)

Alinha os itens no meio da altura disponível.

.container {
  display: flex;
  height: 200px; 
  align-items: center;
}
1
2
3

Flex-End (Base / Fundo)

Empurra os itens para a parte de baixo.

.container {
  display: flex;
  height: 200px;
  align-items: flex-end;
}

5. Centralização Total

Combinando horizontal e vertical para centralizar um elemento perfeitamente.

.container {
  display: flex;
  height: 300px;
  justify-content: center; /* Horizontal */
  align-items: center;     /* Vertical */
}

Casos de Uso Reais

Como aplicamos isso no dia a dia?

Caso 1: Fileira de Cards com Space-Between

Aqui usamos justify-content: space-between. Ele joga o primeiro card para o canto esquerdo, o último para o direito e distribui o espaço restante no meio, criando um grid perfeito sem margens laterais sobrando.

.lista-cards {
  display: flex;
  justify-content: space-between;
  /* gap não é necessário aqui pois o space-between cuida disso */
}

Caso 2: Layout Alternado (Zig-Zag)

Muito comum em sites de apresentação. No segundo container, usamos flex-direction: row-reverse para inverter a ordem visual (Imagem na esquerda) sem mexer no HTML.

/* Container Normal */
.container { display: flex; flex-direction: row; }

/* Container Invertido */
.container-invertido { display: flex; flex-direction: row-reverse; }

Caso 3: Navbar (Logo e Links)

O padrão clássico de menus: Logo em uma ponta, links na outra. Conseguimos isso com justify-content: space-between.

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Caso 4: Card de Produto (Vertical)

Dentro de um card, geralmente queremos os itens um embaixo do outro (Coluna). Usamos justify-content: space-between para garantir que o botão sempre fique no final do card, mesmo que o texto seja curto.

.card-produto {
  display: flex;
  flex-direction: column;
  justify-content: space-between;
  height: 350px;
}
